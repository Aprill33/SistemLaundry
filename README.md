#  Sistem Informasi Laundry

## Deskripsi
Sistem Informasi Laundry adalah aplikasi desktop berbasis C# yang dirancang untuk membantu pengelolaan operasional usaha laundry secara terkomputerisasi.
Aplikasi ini mendukung pengelolaan data pelanggan, layanan laundry, transaksi, serta pembayaran dengan pembagian hak akses antara Admin dan Kasir.

Sistem ini bertujuan untuk menggantikan proses pencatatan manual yang berisiko menimbulkan kesalahan, kehilangan data, dan kesulitan dalam pencarian informasi. Dengan sistem ini, seluruh data tersimpan terpusat di database dan dapat diakses sesuai dengan peran pengguna.

## Fitur
* Login multi-role (Admin & Kasir) dengan validasi akun

* Pengelolaan data pelanggan dan layanan laundry

* Transaksi laundry multi-layanan

* Pembayaran dengan perhitungan otomatis

* Monitoring status laundry dan transaksi

* Manajemen akun dan hak akses pengguna
